---
description: Commit et push des modifications du site-compagnon vers GitHub
---

// turbo-all

1. Vérifier l'état du dépôt
```
git status
```

2. Ajouter tous les fichiers modifiés
```
git add .
```

3. Créer un commit avec un message descriptif en français (adapter selon les modifications)
```
git commit -m "Mise à jour site : [décrire les modifications]"
```

4. Pousser vers GitHub
```
git push origin main
```

5. Confirmer que le push a réussi et rappeler à l'utilisateur que GitHub Pages met 1-3 minutes à se déployer.
